# Bash and linux basic commands

- Where am i?
  - `pwd`

- How to create a Dir?
  - `mkdir name`

- How to create a file?
  - `touch file.format`

- How to change dir
  - `cd folder`

- Move back on a dir
  - `cd ..`

- My username
  - `whoami`

- OS Information
  - `uname -srm`

- List files in a dir
  - `ls`

- Remove file
  - `rm file`

- Move file
  - `mv file location`

- Rename file
  - `mv old_name new_name`
